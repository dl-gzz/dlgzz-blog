## 功能概览

独立工作者提供了丰富的功能来帮助您高效工作和生活。

## 内容管理

### 文章编写

强大的富文本编辑器，支持：

- ✍️ Markdown 语法
- 🖼️ 图片上传
- 📊 表格和图表
- 🔗 链接和引用
- 💾 自动保存

### 分类和标签

组织您的内容：

<Tabs items={['分类', '标签', '搜索']}>
<Tab value="分类">
使用分类建立内容的层级结构：
- 工作
- 学习
- 生活
- 其他
</Tab>

<Tab value="标签">
使用标签快速筛选内容：
- #重要
- #待办
- #灵感
- 自定义标签
</Tab>

<Tab value="搜索">
全文搜索功能：
- 快速查找内容
- 支持关键词高亮
- 历史搜索记录
</Tab>
</Tabs>
