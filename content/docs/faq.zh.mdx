## 账号相关

<Accordions>
<Accordion title="如何注册账号？">

访问[注册页面](/auth/register)，填写邮箱和密码即可注册。

支持的注册方式：
- 📧 邮箱注册
- 🔐 Google 账号（即将支持）
- 🐙 GitHub 账号（即将支持）

</Accordion>

<Accordion title="忘记密码怎么办？">

1. 访问[找回密码](/auth/forgot-password)页面
2. 输入注册邮箱
3. 查收重置邮件
4. 点击链接设置新密码

</Accordion>

<Accordion title="如何修改个人信息？">

进入[个人设置](/settings/profile)页面进行修改：
- 头像
- 昵称
- 个人简介
- 联系方式

</Accordion>

<Accordion title="可以注销账号吗？">

可以。在[账号设置](/settings/account)底部找到"注销账号"选项。

<Callout type="warn">
注销后数据将无法恢复，请谨慎操作
</Callout>

</Accordion>
</Accordions>

## 功能使用

<Accordions>
<Accordion title="如何创建新项目？">

两种方式：

**方式一：** 点击页面右上角的"新建项目"按钮

**方式二：** 使用快捷键 `Cmd/Ctrl + N`

填写项目信息后点击"创建"即可。

</Accordion>

<Accordion title="支持哪些内容格式？">

支持的格式：
- ✍️ Markdown
- 📝 富文本
- 🖼️ 图片（JPG、PNG、GIF、WebP）
- 📹 视频（即将支持）
- 📎 附件（PDF、Word、Excel 等）

</Accordion>

<Accordion title="如何导入历史数据？">

<Steps>
### 准备数据

将数据整理为 Markdown 或 CSV 格式

### 访问导入页面

进入[设置 > 数据导入](/settings/import)

### 上传文件

选择要导入的文件

### 映射字段

确认数据字段对应关系

### 开始导入

等待导入完成
</Steps>

</Accordion>

<Accordion title="数据安全吗？">

我们非常重视数据安全：

- 🔐 **数据加密** - 传输和存储都加密
- 💾 **自动备份** - 每日自动备份
- 🛡️ **权限控制** - 严格的访问控制
- 📜 **审计日志** - 完整的操作记录

</Accordion>
</Accordions>

<Accordion title="页面加载很慢怎么办？">

尝试以下方法：

1. **清除缓存** - 清除浏览器缓存
2. **更换网络** - 尝试更换网络环境
3. **检查插件** - 禁用浏览器插件
4. **更新浏览器** - 使用最新版浏览器
