## API 集成

### 获取 API 密钥

<Steps>
### 进入设置页面

访问 [设置 > API](/settings/api)

### 生成密钥

点击"生成新密钥"按钮

### 保存密钥

<Callout type="warn">
密钥只显示一次，请妥善保存
</Callout>
</Steps>

### 使用 API

```typescript
// 示例：获取文章列表
const response = await fetch('https://api.dlgzz.com/articles', {
  headers: {
    'Authorization': `Bearer YOUR_API_KEY`
  }
});

const articles = await response.json();
```

## 自动化工作流

### Webhook 配置

设置 Webhook 接收事件通知：

1. **新文章发布** - 自动通知订阅者
2. **任务完成** - 触发下一步流程
3. **数据更新** - 同步到其他系统

### 定时任务

创建定时任务自动执行：

- 📅 每日总结
- 📊 周报生成
- 🔔 提醒通知
