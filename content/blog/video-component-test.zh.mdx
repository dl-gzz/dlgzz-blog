## 特性说明

这个视频组件具有以下特性：

- ✅ 响应式设计
- ✅ 平滑的打开/关闭动画
- ✅ 支持明暗主题切换
- ✅ 点击背景关闭
- ✅ YouTube 视频嵌入

## 使用方法

在 MDX 文件中导入组件：

```jsx
import { HeroVideoDialog } from "@/components/blog/hero-video-dialog"
```

然后在文章中使用：

```jsx
<HeroVideoDialog
  videoSrc="你的YouTube视频链接"
  thumbnailSrc="你的缩略图链接"
  thumbnailAlt="视频描述"
/>
```
